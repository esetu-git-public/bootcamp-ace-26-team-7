# Team Roadmap & Roles

This roadmap outlines team responsibilities, sprint tasks, and sprint status.

## Team Roles
- **Team Lead**: Coordination and code review.
- **Scrum Master**: Agile sprint tracking, Github issue management.
- **Product Owner**: Product requirements & documentation.
- **Business Owner**: Strategic goals & product alignment.
- **Developers / ML Engineers**: Implementing the core computer vision models and pipelines.

## Sprint Roadmap (15 Days)
- **Phase 1: Setup & Initialization (Days 1-3)**
  - [x] Sprint planning & setup
  - [x] Initial login UI implementation
  - [x] Basic workspace folder structure setup
- **Phase 2: Data Exploration & Pipeline (Days 4-7)**
  - [ ] Exploratory Data Analysis (EDA)
  - [ ] Data preprocessing & splitting scripts
  - [ ] Custom PyTorch Dataset definition
- **Phase 3: Model Development & Training (Days 8-11)**
  - [ ] Baseline CNN & Transfer learning ResNet models
  - [ ] Training loops with early stopping
- **Phase 4: Evaluation & UI Integration (Days 12-15)**
  - [ ] Model evaluation and results visualization
  - [ ] Streamlit interface integration with the trained model
